## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, button padding-x |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, page header margins |
| xl | 32px | Layout gaps between sidebar and main |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level top/bottom spacing |

Exceptions: none

## Color
